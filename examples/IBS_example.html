IBS/IBD Neighbor Computation Example¶

This example shows how to run computeIBSpbwt to produce the IBS neighbors file required by GRiD Step 5 (haplotype inference). The script targets LPA KIV-2 on chromosome 6 in hg19, but the same pattern applies to any locus — adjust CHR, FOCAL_BP, and the matching BGEN/genetic-map files accordingly.

Before running, compile computeIBSpbwt.cpp and obtain the required input files. See the Computing IBS/IBD Neighbors page for full instructions.

IBS_example.sh — SLURM/local script for computing IBS neighbors¶
  1#!/bin/bash
  2# =============================================================================
  3# GRiD — IBS/IBD Neighbor Computation
  4# LPA KIV-2 (hg38, chr6) using 1000 Genomes phased panel
  5#
  6# Pre-requisite for GRiD Step 7 (haplotype inference).
  7# Runs computeIBSpbwt from Hujoel et al. (2026) Nature.
  8# Source: https://github.com/mhujoel/STRs/blob/main/cpp_files/computeIBSpbwt.cpp
  9#
 10# This script:
 11#   1. Downloads the 1000G phased VCF for chr6 (no login required)
 12#   2. Extracts the LPA region and converts to BGEN v1.2 via qctool
 13#   3. Downloads the Eagle hg38 genetic map
 14#   4. Runs computeIBSpbwt to produce the IBS neighbors file
 15#
 16# Usage:
 17#   bash   IBS_example.sh          # local run
 18#   sbatch IBS_example.sh          # SLURM submission
 19#
 20# Output: ibs_neighbors_chr6.tsv.gz
 21#   Pass to GRiD via compute_haploid_genotypes.ibs_output in your config.
 22# =============================================================================
 23#SBATCH --job-name=grid_IBS
 24#SBATCH --output=logs/IBS_%j.out
 25#SBATCH --error=logs/IBS_%j.err
 26#SBATCH --time=4:00:00
 27#SBATCH --cpus-per-task=16
 28#SBATCH --mem=64G
 29
 30set -euo pipefail
 31
 32# -----------------------------------------------------------------------
 33# Dependency check
 34# -----------------------------------------------------------------------
 35MISSING=()
 36for cmd in qctool wget samtools; do
 37    command -v "$cmd" &>/dev/null || MISSING+=("$cmd")
 38done
 39if [[ ${#MISSING[@]} -gt 0 ]]; then
 40    echo "ERROR: missing required tools: ${MISSING[*]}"
 41    echo "  qctool:   https://www.well.ox.ac.uk/~gav/qctool_v2/"
 42    echo "  samtools: conda install -c bioconda samtools"
 43    exit 1
 44fi
 45
 46# -----------------------------------------------------------------------
 47# Paths — set WORK_DIR or override individual variables
 48# -----------------------------------------------------------------------
 49WORK_DIR="${WORK_DIR:-$(pwd)/1000G_grid_run}"     # match 1000G_example.sh
 50
 51DATA_DIR="$WORK_DIR/data"
 52LOG_DIR="$WORK_DIR/logs"
 53mkdir -p "$DATA_DIR" "$LOG_DIR"
 54
 55COMPUTE_IBS="${COMPUTE_IBS:-$(pwd)/computeIBSpbwt}"   # compiled binary path
 56
 57# -----------------------------------------------------------------------
 58# 1000 Genomes public URLs
 59# -----------------------------------------------------------------------
 60PHASED_VCF_URL="https://ftp.1000genomes.ebi.ac.uk/vol1/ftp/data_collections/1000G_2504_high_coverage/working/20220422_3202_phased_SNV_INDEL_SV/1kGP_high_coverage_Illumina.chr6.filtered.SNV_INDEL_SV_phased_panel.vcf.gz"
 61GENETIC_MAP_URL="https://alkesgroup.broadinstitute.org/Eagle/downloads/tables/genetic_map_hg38_withX.txt.gz"
 62
 63# -----------------------------------------------------------------------
 64# Parameters
 65# -----------------------------------------------------------------------
 66CHR=6
 67FOCAL_BP=160626361       # center of LPA KIV-2 VNTR region (hg38)
 68LPA_REGION="chr6:160605062-160647661"
 69NUM_NEIGHBORS=200
 70THREADS="${SLURM_CPUS_PER_TASK:-$(nproc)}"
 71
 72PHASED_VCF="$DATA_DIR/1kGP_chr6_phased.vcf.gz"
 73LPA_VCF="$DATA_DIR/1kGP_chr6_LPA_phased.vcf.gz"
 74BGEN_FILE="$DATA_DIR/1kGP_chr6_LPA_phased.bgen"
 75SAMPLE_FILE="$DATA_DIR/1kGP_chr6_LPA_phased.sample"
 76GENETIC_MAP="$DATA_DIR/genetic_map_hg38_chr6.txt"
 77OUTPUT_FILE="$DATA_DIR/ibs_neighbors_chr6.tsv.gz"
 78
 79timestamp() { date '+%Y-%m-%d %H:%M:%S'; }
 80log() { echo "[$(timestamp)] $*" | tee -a "$LOG_DIR/IBS.log"; }
 81
 82log "================================================="
 83log "  IBS/IBD Neighbor Computation"
 84log "  Locus: LPA KIV-2 ($LPA_REGION, hg38)"
 85log "  Threads: $THREADS"
 86log "================================================="
 87
 88# Verify computeIBSpbwt binary
 89if [[ ! -x "$COMPUTE_IBS" ]]; then
 90    echo "ERROR: computeIBSpbwt binary not found or not executable at: $COMPUTE_IBS"
 91    echo "  Download source: https://github.com/mhujoel/STRs/blob/main/cpp_files/computeIBSpbwt.cpp"
 92    echo "  Then compile and set: export COMPUTE_IBS=/path/to/computeIBSpbwt"
 93    exit 1
 94fi
 95
 96# -----------------------------------------------------------------------
 97# Phase 1 — Download 1000G phased VCF for chr6
 98#
 99# The full chr6 phased panel is ~5 GB. We stream only the LPA window
100# into a local subset VCF using tabix (no full download required).
101# -----------------------------------------------------------------------
102if [[ ! -f "$LPA_VCF" ]]; then
103    log "Streaming LPA region from 1000G phased VCF..."
104
105    # Fetch index alongside to enable remote tabix streaming
106    wget -q -c -O "${PHASED_VCF}.tbi" "${PHASED_VCF_URL}.tbi"
107
108    # Pull only the LPA window — ~1 MB vs 5 GB for full chr6
109    tabix -h "$PHASED_VCF_URL" "$LPA_REGION" \
110        | bgzip > "$LPA_VCF"
111    tabix -p vcf "$LPA_VCF"
112
113    log "LPA phased VCF ready: $LPA_VCF"
114else
115    log "LPA phased VCF already present — skipping."
116fi
117
118# -----------------------------------------------------------------------
119# Phase 2 — Convert phased VCF → BGEN v1.2
120#
121# qctool outputs BGEN v1.2, layout 2, 16-bit, phased — the format
122# required by computeIBSpbwt.
123# -----------------------------------------------------------------------
124if [[ ! -f "$BGEN_FILE" ]]; then
125    log "Converting phased VCF to BGEN v1.2..."
126    qctool \
127        -g "$LPA_VCF" \
128        -og "$BGEN_FILE" \
129        -os "$SAMPLE_FILE" \
130        -ofiletype bgen_v1.2 \
131        -bgen-bits 16 \
132        -bgen-permitted-input-rounding-error 0 \
133        >> "$LOG_DIR/qctool.log" 2>&1
134    log "BGEN file ready: $BGEN_FILE"
135else
136    log "BGEN file already present — skipping."
137fi
138
139# -----------------------------------------------------------------------
140# Phase 3 — Download Eagle genetic map for hg38
141# -----------------------------------------------------------------------
142if [[ ! -f "$GENETIC_MAP" ]]; then
143    log "Downloading Eagle hg38 genetic map..."
144    wget -q -O - "$GENETIC_MAP_URL" \
145        | gunzip \
146        | awk 'NR==1 || $1=="chr6"' \
147        > "$GENETIC_MAP"
148    log "Genetic map ready: $GENETIC_MAP"
149else
150    log "Genetic map already present — skipping."
151fi
152
153# -----------------------------------------------------------------------
154# Phase 4 — Run computeIBSpbwt
155# -----------------------------------------------------------------------
156log "Computing IBS/IBD neighbors..."
157log "  CHR=$CHR  FOCAL_BP=$FOCAL_BP  NEIGHBORS=$NUM_NEIGHBORS  THREADS=$THREADS"
158
159"$COMPUTE_IBS" \
160    "$CHR"          \
161    "$FOCAL_BP"     \
162    "$BGEN_FILE"    \
163    "$SAMPLE_FILE"  \
164    "$GENETIC_MAP"  \
165    "$NUM_NEIGHBORS"\
166    "$THREADS"      \
167    "$OUTPUT_FILE"
168
169log "================================================="
170log "  Done. IBS neighbors written to:"
171log "    $OUTPUT_FILE"
172log ""
173log "  Add to your GRiD config:"
174log "    compute_haploid_genotypes:"
175log "      run: True"
176log "      ibs_output: \"$OUTPUT_FILE\""
177log "================================================="

Key Parameters¶

Edit the Paths and Parameters sections at the top of the script before submitting:

Variable

Description

COMPUTE_IBS

Path to the compiled computeIBSpbwt binary

BGEN_FILE

Phased BGEN v1.2 file for the chromosome of interest

SAMPLE_FILE

Oxford-format .sample file matching the BGEN

GENETIC_MAP

Eagle genetic map for the chromosome (download from Eagle website)

OUTPUT_FILE

Output path; use .tsv.gz for automatic gzip compression

FOCAL_BP

Base pair position at the center of the VNTR region (hg19: 160690000)

NUM_NEIGHBORS

Number of IBS neighbors per haplotype (recommended: 200)

THREADS

CPU threads; match to --cpus-per-task in the SLURM header

Usage¶

Submit to SLURM:

sbatch IBS_example.sh

Or run locally:

bash IBS_example.sh

Passing Output to GRiD¶

Once complete, point to the output file in your GRiD config:

compute_haploid_genotypes:
  run: True
  output_file_prefix: "haploid_genotypes"
  ibs_output: "path/to/ibs_neighbors_chr6.tsv.gz"
  min_neighbors: 1
  max_neighbors: 10
  n_iters: 100
