GRiD — Genomic Repeat inference from Depth
==========================================

A Python pipeline for haplotype-resolved VNTR copy number estimation from short-read whole-genome sequencing data.
Developed for and validated on the KIV-2 VNTR in the *LPA* gene; the core depth-normalization framework is applicable to any large VNTR locus.
